Soil Moisture Retrieval using Sentinel-1

Paschim & Purba Bardhaman District · SAR-based Environmental Monitoring

Sentinel-1 SAR ESA SNAP ArcGIS Pro QGIS

Objective

Retrieve and map soil moisture distribution across Bardhaman district using Synthetic Aperture Radar data from Sentinel-1. Validate satellite-derived estimates with in-situ observations to establish robust methodology for agricultural water management and drought monitoring applications.

Data & Tools

Satellite Data
Sentinel-1 GRD (VV/VH polarization)
Ancillary Data
SRTM/AWS DEM, in-situ soil moisture samples
Processing Tools
ESA SNAP, ArcGIS Pro, QGIS

Methodology

Data Acquisition
Downloaded Sentinel-1 GRD for study period, subset to district AOI, applied orbit files and thermal noise removal
SAR Processing
Radiometric calibration to sigma0, Refined Lee speckle filtering, Range-Doppler terrain correction using DEM
Feature Extraction
Converted to backscatter decibel (dB), derived VV/VH polarization ratios, integrated NDVI as ancillary data
Model Development
Implemented soil moisture retrieval using empirical models with ancillary data integration
Validation
Collocated in-situ measurements with satellite data, created final soil moisture maps in ArcGIS

Validation Metrics

RMSE Analysis
Bias & R²
Classification
Confusion Matrix
Uncertainty
Error Maps

Deliverables

  • Soil moisture rasters with spatial coverage
  • District-level soil moisture distribution maps
  • Comprehensive validation report with statistical analysis
  • Automated processing scripts (SNAP graph XML workflows)

Challenges & Solutions

  • Sparse in-situ data → Implemented temporal matching techniques
  • Vegetation cover signal interference → Utilized VH/VV ratio and incorporated NDVI
  • Model uncertainty → Developed comprehensive error propagation framework

+

Advanced Flood Hazard Mapping

Kamrup Metropolitan · Multi-sensor Integration & Impact Assessment

Google Earth Engine Sentinel-1 & 2 SAR/Optical Fusion Population Analysis Night-time Lights

Objective

Generate high temporal-resolution flood hazard maps using multi-sensor satellite data, refine detection accuracy with spectral indices, and assess socio-environmental impacts including population displacement and settlement pattern changes through integrated spatial analysis.

Data Sources

SAR Data
Sentinel-1 C-band (VV & VH polarization)
Optical Data
Sentinel-2 multispectral imagery
Socio-economic
Night-time lights, population density, census layers
Platform
Google Earth Engine, QGIS/ArcGIS

Processing Workflow

Data Preparation
Ingested AOI and temporal window in GEE, created Sentinel-1 time series with speckle reduction
Flood Detection
Applied threshold-based and change detection on SAR backscatter comparing pre-flood baseline with flood imagery
Optical Refinement
Computed NDWI, MNDWI, and SWIR thresholds from Sentinel-2 to mask false positives (shadows, built-up)
Severity Analysis
Classified flood severity (transient, sustained) using duration and recurrence frequency patterns
Impact Assessment
Overlaid flood maps with settlement/census for exposure analysis, analyzed night-time lights for displacement trends

Key Outputs

  • Multi-temporal flood extent maps with daily to weekly resolution
  • Flood severity classification maps (transient vs. sustained flooding)
  • Population exposure statistics and vulnerability assessment
  • Migration/displacement hotspot identification maps
  • Complete Google Earth Engine processing scripts

Validation

  • Cross-validation with documented flood events and local reports
  • Comparison against high-resolution imagery for spatial accuracy
  • Precision/recall analysis for flood detection performance
  • Temporal consistency checks across multiple flood events

Land Subsidence Visualization — Coal Mining Areas

Paschim Bardhaman · InSAR Time Series Analysis

Sentinel-1 InSAR ESA SNAP SBAS/PSInSAR Geodetic Analysis Hazard Mapping

Project Objective

Detect, quantify, and visualize land subsidence caused by underground coal mining using advanced Interferometric Synthetic Aperture Radar techniques. Employed Persistent Scatterer and Small Baseline Subset InSAR methodologies to generate high-precision displacement maps for infrastructure risk assessment.

Technical Infrastructure

SAR Data
Sentinel-1 IW SLC time series (C-band)
Ancillary
High-resolution DEM, mining lease boundaries
Software
SNAP, GMTSAR/ISCE/StaMPS, SARscape
Visualization
ArcGIS Pro, Python (matplotlib, plotly)

InSAR Processing Workflow

Data Stack Preparation
Collected comprehensive time series of Sentinel-1 SLC acquisitions (30+ scenes)
Coregistration
Performed precise image coregistration, generated interferogram stack
Phase Processing
Removed topographic phase using DEM, applied adaptive filtering
PS/SBAS Analysis
Identified persistent scatterers, applied SBAS inversion, estimated atmospheric contributions
Phase Unwrapping
Applied network flow unwrapping, quality-controlled using coherence thresholds
Time Series Inversion
Estimated displacement velocity and time-dependent deformation, filtered outliers
Visualization
Geocoded results, generated subsidence rate maps, animated time series

Visualization Outputs

  • Subsidence velocity maps (mm/year) with color-coded deformation rates
  • Cumulative displacement time series for critical zones
  • Cross-sectional profiles showing subsidence bowls
  • Animated time-lapse showing temporal evolution
  • Hazard zonation maps identifying critical subsidence areas
  • 3D terrain visualization with overlaid subsidence magnitude
  • Infrastructure risk overlay (buildings, roads, mining boundaries)

Validation

Precision
mm-level
Temporal Res.
12-day
Coverage
District-wide

Challenges

  • Atmospheric phase delays → Mitigated through multi-temporal stacking and tropospheric modeling
  • Decorrelation in vegetated areas → Focused on urban/bare earth persistent scatterers
  • Large displacement gradients → Applied enhanced unwrapping algorithms
  • Computational intensity → Implemented parallel processing and cloud computing
